Web Interface

Neodyme includes a full web interface accessible at http://localhost:3551 (or your configured port). It provides dashboard access for players, moderators, admins, and developers.

Accessing the Dashboard

  1. Open your browser to http://localhost:3551
  2. Register or login with your account
  3. Navigate using the sidebar menu
  4. Role-specific panels appear automatically based on your account role

To promote your first account to admin, use the server console:

/admin set "YourUsername" admin

Player Dashboard

Available to all registered users.

Profile

  • View and edit display name
  • View account ID and email
  • Change password

Friends

  • View friends list with online/offline status
  • Send friend requests by username
  • Accept or reject incoming requests
  • Remove friends and block/unblock users

Sessions

  • View all active sessions (devices)
  • Terminate individual sessions
  • Terminate all sessions (logout everywhere)

Item Shop

  • Browse current shop items (Featured & Daily sections) with rarity-coloured cards
  • View item details, prices, and "Already Owned" badge
  • Purchase items with V-Bucks (with confirmation dialogs)
  • Support a creator: enter a creator code in the sidebar — commission is automatically credited on purchase

V-Bucks

  • View current V-Bucks balance
  • Purchase V-Bucks via a multi-step flow: Package → Payment → Review → Done
  • Enter a creator code during checkout to support a creator (step 2 of the flow)
  • View purchase history
  • Refund purchases

Creator Code

  • View your current creator code
  • Request a new creator code (3–16 chars, alphanumeric + _ -)
  • View request status (pending/approved/rejected)
  • Earn commission when other players support your code during V-Bucks or item shop purchases
  • Commission tracking and earnings history

Support Tickets

  • Create support tickets with subject, message, and priority (low/medium/high)
  • View your ticket history and reply to open tickets
  • Track ticket status (open, in_progress, resolved, closed)
  • Maximum 3 open tickets at a time

Moderator Panel

Available to Moderator, Admin, and Owner roles.

Player Management

  • Search players by username
  • View detailed player information (account ID, email, role, ban status)
  • Ban players (temporary or permanent) with reason
  • Unban players and view ban history

Ticket Management

  • View all support tickets with filters (status, priority)
  • Assign/unassign tickets, reply as moderator, change status/priority
  • Delete tickets
  • Ticket statistics overview (total, open, unassigned, high priority)

Creator Code Management

  • View pending creator code requests — approve or reject with notes
  • View all active creator codes
  • Toggle code active/inactive, delete codes
  • Creator code statistics (total codes, earnings, uses)

Commands NEW 1.2.4

  • Execute moderation commands directly from the panel (ban, unban, account lookup)
  • Only commands available to the Moderator role are shown

Admin Panel

Available to Admin and Owner roles. Includes all Moderator features plus:

User Management

  • View all user accounts
  • Change user roles (Player, Founder, Helper, Moderator, Admin, Owner)
  • Ban/unban users from the admin panel

Audit Log

  • View complete audit log of all administrative actions
  • Filter by action type, administrator, date range
  • Audit log summary (action counts over 24h)

Developer Panel

Available to Developer role only. Includes all Admin features plus:

Configuration Editor

Full control over all server.properties settings organized by category: Core Server, Debug, Features, XMPP, Database, Redis, Security, Rate Limiting, Authentication, CSRF, Web Tokens, Creator Codes, Events, Maintenance, SSL/HTTPS, and more.

Configuration Files

  • View and edit config files (shop_config.json, xp_config.json, etc.)
  • JSON editor with validation

Server Statistics

  • Detailed server metrics (uptime, memory, CPU)
  • API request statistics and database type/connection status

SSL/HTTPS Status

  • Current protocol (HTTP/HTTPS)
  • Certificate file status (exists, size, modification date)

Live Monitor NEW 1.2.4

  • Real-time CPU usage chart (updated every 3 seconds)
  • Real-time RAM usage chart
  • Active player count chart
  • Start/stop polling from the dashboard

Console Logs NEW 1.2.4

  • Read server log files directly in the browser
  • Filter by log level (INFO, WARN, ERROR, SUCCESS)
  • Auto-refresh every 5 seconds
  • Color-coded log entries by severity

Plugin Manager NEW 1.2.4

  • View all loaded plugins (name, version, author, description)
  • Load, unload, and reload individual plugins from the UI
  • Browse and edit plugin configuration files directly from the browser
  • Real-time feedback on plugin status changes
