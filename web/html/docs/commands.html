Console Commands

Commands are entered in the server terminal. All commands start with /.

General

CommandDescription
/help [page]List available commands (paginated, 10 per page)
/versionShow server version
/uptimeShow server uptime
/memoryShow heap memory usage
/healthFull health check (memory, tokens, platform)
/readyCheck if all services are operational
/clearClear console and print ASCII logo
/stopGracefully shut down the server

/reload IMPROVED 1.2.4

CommandDescription
/reloadReload config + plugins + shop in one shot
/reload configReload configuration from server.properties only
/reload pluginsUnload and reload all plugins
/reload shopRe-initialize shop rotation

/data - Database Migration NEW 1.2.4

CommandDescription
/data migrate json mongodbMigrate all data from JSON files to MongoDB
/data migrate mongodb jsonMigrate all data from MongoDB to JSON files

Flags:

  • --dry-run - Preview what would be migrated, no data is written
  • --confirm - Skip the confirmation prompt and run immediately
# Preview migration
/data migrate json mongodb --dry-run

# Run with auto-confirm
/data migrate json mongodb --confirm

/test - Test Suite NEW 1.2.4

CommandDescription
/test allRun all test suites
/test accountAccount + database read tests
/test pluginsPlugin system tests
/test serverConfig + token service tests
/test databaseDatabase connection tests

/diagnostic NEW 1.2.4

Prints a full server diagnostic report: OS, CPU, RAM, uptime, database type, active services, token counts, and loaded plugins.

/diagnostic

/account

CommandDescription
/account create <email> <password> <name>Create a new account
/account info <username>View account details (role, ban, V-Bucks)
/account list [page]List all accounts (paginated)

/admin

CommandDescription
/admin set <user> <role>Set role: 0=Player 1=Mod 2=Dev 3=Admin 4=Owner 5=Server
/admin listList all staff members grouped by role
/admin ban <user> [reason]Ban a player
/admin unban <user>Unban a player
/admin info <user>View player information

/ban

CommandDescription
/ban add <user> <reason> [duration]Ban a player (duration: 2h, 7d, 1m, or omit for permanent)
/ban remove <user>Unban a player
/ban info <user>View ban details and time remaining
/ban listList all banned players

/shop

CommandDescription
/shop rotateForce shop rotation immediately
/shop infoShow current daily and featured items
/shop statusShow last/next rotation times

/plugins

CommandDescription
/plugins listList loaded plugins
/plugins load <name|all>Load plugin(s)
/plugins unload <name|all>Unload plugin(s)
/plugins reload <name|all>Reload plugin(s)
/plugins info <name>Show plugin details
/plugins store list [page]Browse plugin store
/plugins store search <query>Search plugins
/plugins store install <id>Install plugin from store
/plugins store update <id>Update installed plugin
/plugins store uninstall <id>Uninstall plugin
/plugins store refreshRefresh store cache

/backup

CommandDescription
/backup createCreate a manual backup
/backup listList all backups with size and date
/backup statsBackup system statistics
/backup restore <name>Restore a backup (requires /confirm restore)
/backup delete <name>Delete a backup (requires /confirm delete)
/backup cleanupRemove old backups based on retention policy

/tokens

CommandDescription
/tokens statsShow access/refresh/client token counts
/tokens cleanupRemove expired tokens
/tokens revoke <username|all>Revoke tokens for a user or all users
/tokens user <username>Show token details for a user

/maintenance NEW 1.2.4

CommandDescription
/maintenance onEnable maintenance mode — blocks new player logins
/maintenance offDisable maintenance mode

/broadcast NEW 1.2.4

CommandDescription
/broadcast <message>Send a message to all connected players

Miscellaneous

CommandDescription
