Console Commands

Commands are entered in the server terminal. All commands start with /.

General

CommandDescription
/help [page]List available commands (paginated, 10 per page)
/versionShow server version
/uptimeShow server uptime
/memoryShow heap memory usage
/healthFull health check (memory, tokens, platform)
/readyCheck if all services are operational
/clearClear console and print ASCII logo
/stopGracefully shut down the server

/reload

CommandDescription
/reloadReload config + plugins + shop in one shot
/reload configReload configuration from server.properties only
/reload pluginsUnload and reload all plugins
/reload shopRe-initialize shop rotation

/data - Database Migration

CommandDescription
/data migrate json mongodbMigrate all data from JSON files to MongoDB
/data migrate mongodb jsonMigrate all data from MongoDB to JSON files

Flags:

  • --dry-run - Preview what would be migrated, no data is written
  • --confirm - Skip the confirmation prompt and run immediately
# Preview migration
/data migrate json mongodb --dry-run

# Run with auto-confirm
/data migrate json mongodb --confirm

/test - Test Suite

CommandDescription
/test allRun all test suites
/test accountAccount + database read tests
/test pluginsPlugin system tests
/test serverConfig + token service tests
/test databaseDatabase connection tests

/diagnostic

Prints a full server diagnostic report: OS, CPU, RAM, uptime, database type, active services, token counts, and loaded plugins.

/diagnostic

/account

CommandDescription
/account create <email> <password> <name>Create a new account
/account info <username>View account details (role, ban, V-Bucks)
/account list [page]List all accounts (paginated)

/give

CommandDescription
/give vbucks <username> <amount>Give V-Bucks to a player
/give item <username> <templateId>Give a cosmetic item to a player's locker

The templateId must include the type prefix and the item ID, separated by :. For example:

/give item JojoM AthenaCharacter:CID_A_248_Athena_Commando_M_ZombieElastic_E
/give item JojoM AthenaBackpack:Backpack_Titanium
/give item JojoM AthenaDance:EID_BoogieDown
/give item JojoM AthenaPickaxe:Pickaxe_ID_010_Lockjaw

Supported item types:

  • AthenaCharacter - Outfits (skins)
  • AthenaBackpack - Back Blings
  • AthenaPickaxe - Harvesting Tools
  • AthenaGlider - Gliders
  • AthenaDance - Emotes
  • AthenaItemWrap - Wraps
  • AthenaSkyDiveContrail - Contrails
  • AthenaMusicPack - Music
  • AthenaLoadingScreen - Loading Screens
  • AthenaBattleBus, AthenaVehicleCosmetics, AthenaPet, AthenaPetCarrier, AthenaSpray, AthenaToy, AthenaEmoji

If the player already owns the item, the command will warn you instead of adding a duplicate.

/plugins

CommandDescription
/plugins listList loaded plugins

/plugins unload <name|all>Unload plugin(s)
/plugins reload <name|all>Reload plugin(s)

/assets

Manage how Neodyme serves /images/* assets - see Game Files → Asset Pipeline for the full guide.

CommandDescription
/assets statusShow current mode, index entries, missing assets, local disk usage
/assets mode <online|local>Switch serving mode (online → local triggers automatic download of missing files)
/assets list [--tag <tag>]List remote assets, filterable by tag
/assets info <path>Show local + remote details for a specific asset
/assets diagnose <path>Explain how a /images/<path> request resolves
/assets install [path]Download all missing assets, or one specific asset (sizes validated post-download)
/assets uninstall [path]Remove a single asset (or all managed assets, requires /confirm assets-uninstall)
/assets cleanRemove orphan local files no longer in the index (requires /confirm assets-clean)
/assets verifyAudit local files vs the index (missing, zero-byte, size mismatches, orphans)
/assets reloadForce-reload content/assets-index.json from disk
/assets refreshForce-refresh the remote manifest cache

/backup

CommandDescription

/backup restore <name>Restore a backup (requires /confirm restore)
/backup delete <name>Delete a backup (requires /confirm delete)
/backup cleanupRemove old backups based on retention policy
