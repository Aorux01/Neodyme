Getting Started

Everything you need to install and run Neodyme.

Neodyme requires Node.js 18+. Download it from nodejs.org.

Installation

  1. Download or clone the latest version of Neodyme.
  2. Run install_packages.bat to install all npm dependencies (only required once).
  3. Launch the server with start.bat.
  4. Open http://localhost:3551 in your browser.

Configuration

All settings are in server.properties. The most important ones:

KeyDefaultDescription
port3551HTTP port the server listens on
protocolhttphttp or https
databaseTypejsonjson or mongodb
xmppEnabletrueEnable XMPP/chat server
autoShopRotationtrueAutomatic daily shop rotation
pluginstrueEnable plugin system
webInterfacetrueEnable web dashboard
maintenanceModefalsePut server in maintenance mode

Environment Variables (.env)

Secrets are stored in .env (auto-generated on first startup). You can override them:

JWT_SECRET=your_jwt_secret_here
GAME_SERVER_SECRET=your_game_server_secret_here
MONGODB_URI=mongodb://localhost:27017/neodyme
REDIS_URL=redis://localhost:6379

First Account

Create your first admin account via the console:

/account create admin@example.com Password123! AdminName
/admin set AdminName owner

Connecting the Game

Point your Fortnite launcher to http://localhost:3551. Configure your DNS or hosts file to redirect Epic's domains to 127.0.0.1.
